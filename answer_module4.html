Topic 1: Knowledge Based Agent
  • What is a knowledge-based agent? How does it work? (3)
  • What is a knowledge-based agent? How does it work? Write an algorithm for Knowledge based agent. (7)
  • What is a knowledge-based agent? Write an algorithm. (7)

A Knowledge-Based Agent (KBA) is an intelligent agent that uses a central repository of information, known as a Knowledge Base (KB), to make decisions and perform actions. Unlike simple reflex agents, it "knows" about its world and can reason to reach a goal.

How it works

A Knowledge-Based Agent operates through a continuous cycle of three main steps:

  • TELL: The agent "tells" the Knowledge Base what it perceives from its environment through sensors, updating its internal model of the world.
  • ASK: The agent "asks" the Knowledge Base what action it should perform based on its current knowledge and its goals.
  • EXECUTE: Once the KB provides a logical conclusion or action, the agent performs that action via its actuators.
Topic 2: Conjunctive Normal Form (CNF)
  • What do you mean by Conjunctive Normal Form? (3)
  • Steps for converting propositional logic to CNF. Convert (P -> Q) -> ((Q -> R) -> (P -> R)). (6)

1. Definition:

CNF is a way of organizing logical formulas where the statement is a conjunction of one or more clauses. In simpler terms, it is an "AND of ORs."

2. Structure:

  • Literals: A variable (e.g., P) or its negation (e.g., ¬P).
  • Clauses: A series of literals connected only by OR (∨) operators.
  • The Formula: Multiple clauses connected together only by AND (∧) operators.

3. Format Example:

(A ∨ B) ∧ (¬B ∨ C ∨ D) ∧ (E)

(Each set of parentheses is a clause, and they are all joined by AND.)

4. Significance:

CNF is the standard format required for Resolution, a powerful inference rule used by Knowledge-Based Agents to perform logical reasoning and prove theorems.

Topic 3: Skolemization
  • Explain the term Skolemization. (3)

Skolemization is a process used in First-Order Logic to remove existential quantifiers (&exists;) from a logical formula while preserving its satisfiability. It is a crucial step in converting logical sentences into Conjunctive Normal Form (CNF) for automated reasoning.

Topic 4: Difference Between Forward and Backward Chaining
  • Differentiate Forward Chaining and Backward Chaining with their algorithms. (9)
  • List six differences between Forward and Backward chaining. (6)
Feature Forward Chaining Backward Chaining
1. Direction Data-driven: It starts from known facts and moves forward to find a conclusion. Goal-driven: It starts from a specific goal (hypothesis) and works backward to find supporting facts.
2. Process It follows the "If-Then" rule: If A is true, then B is true. It asks: "What must be true" to achieve goal G?
3. Strategy It is an antecedent-driven strategy (looking at the "if" part). It is a consequent-driven strategy (looking at the "then" part).
4. Search Uses Breadth-First Search (BFS) to explore all possible paths from facts. Uses Depth-First Search (DFS) to find a specific path to the goal.
5. Suitability Best for systems where there are many facts but few possible goals (e.g., control systems). Best for systems where there are many facts but only a few specific hypotheses to test.
6. Examples Used in Expert Systems for monitoring, planning, and real-time scheduling. Used in Diagnostic Systems (e.g., medical diagnosis or debugging code).
