Alpha Beta Pruning

It is an optimisation technique for the minimax algorithm. Two parameters can be defined as:

  • Alpha (α): Best choice (highest value) we have found in the path of the Maximizer. Initial value = -∞ (negative infinity)
  • Beta (β): Best choice (lowest value) we have found in the path of the Minimizer. Initial value = +∞ (positive infinity)

Condition for alpha-beta pruning: α ≥ β (alpha greater than or equal to beta).
